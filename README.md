# Zento — Social Media Platform

A front-end social media platform built as part of the **CMPS 350 Web Development** course project (Phase 1). The platform is inspired by modern social networks like Twitter and Instagram, designed to be clean, responsive, and fully functional using only core web technologies.

## Overview

Zento is a browser-based social media platform where users can register, log in, create posts, interact with others, and manage their profiles. All data is stored locally in the browser using `localStorage`, with no backend or external libraries involved. The entire interface is built with semantic HTML5, CSS3 (Flexbox & Grid), and Vanilla JavaScript (ES6+).

## Tech Stack

- **HTML5** — Semantic structure using `<nav>`, `<main>`, `<section>`, and more
- **CSS3** — Layouts built entirely with Flexbox and Grid; no frameworks used
- **JavaScript (ES6+)** — DOM manipulation, event handling, and localStorage management
- **Git & GitHub** — Version control and collaboration

## Data Storage

All application data (users, posts, likes, comments) is stored in the browser's `localStorage` as JSON. No backend or database is required — the app works fully offline and persists data between sessions.
